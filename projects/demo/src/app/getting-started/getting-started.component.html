Getting Started

This guide explains how to install, set up and use the "ngx-advanced-material-table" library in your Angular project.

Install Angular Material

Use the Angular CLI's installation schematic to set up your Angular Material project by running the following command:

ng add @angular/material

Install ngx-advanced-material-table

npm i ngx-advanced-material-table

Import NgxAdvancedMaterialTableModule module

Add the NgxAdvancedMaterialTableModule module to the imports of your application module
